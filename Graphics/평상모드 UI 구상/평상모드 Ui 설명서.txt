일단 대강의 틀만을 구상해 봤습니다.


특정버튼을 누르면 우측 메뉴가 뜨도록 하고

화살표를 이용해 선택이 가능하게 하며

메뉴가 떴을때 필드는 그대로 보이도록 합니다.


아이템은 구현할지 안할지 모르겠지만 일단 구상은 해두겠습니다.

아이템 메뉴를 선택하면 주메뉴 위에 아이템 메뉴를 덮어씌우듯이 띄웁니다.
(다른 메뉴도 마찬가지로)


장비와 상태메뉴는 같은 모양을 가집니다.

단, 상태는 정보를 보여주고 장비는 입고 있는 장비를 바꿀수 있으며

장비옆에 주 메뉴에있는 것과 같은 화살표를 출력합니다.

그리고 장비를 선택할 경우 아이템메뉴가 뜨게 됩니다.

아이템 메뉴에서 장비를 선택할경우 바뀌게 되도록 합니다.


저장과 불러오기 메뉴도 같은 모양을 가집니다.

그림에 있는대로 저장했을때 상태정보를 출력하는것이 좋을듯 합니다.

저장공간은 많을수도 있겠지만 5개 정도를 만드는 것이 좋겠습니다.(플레이시간이 짧으니 말이지요.)


설정은 어떤 메뉴가 들어갈지 아직 모르겠습니다만

해상도 조절, B.G.M. 온오프, 효과음 온오프, (인트로 온오프)

같은것들이 있을 수 있겠는데 

내용이 많아지면 화면 전체를 설정에서 쓰도록 하고

내용이 적다면 아이템 메뉴와 비슷한 창을 띄워 

항목을 선택할때 마다 온오프, 또는 해상도가 바뀌게 하는 방법이 있을 수 있겠습니다.


끝내기 메뉴는 선택했을 때 재확인 메시지를 띄우고 Yes와 No 중에 고르도록 합니다.
