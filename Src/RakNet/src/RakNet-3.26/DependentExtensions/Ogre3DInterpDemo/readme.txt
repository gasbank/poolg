Project: Ogre 3D Interpolation Demo

Description: Demonstrates how to lag a client in the past using the interpolation history class, in order to get smooth visuals despite choppy input. Start two instances on the same computer, press 's' on one instance, 'c' on the other. Hold down space to see the actual networking.

Dependencies: Ogre 3D 1.4.5, assmed to have an enviroment variable $(OGRE_HOME) indicating the installation directory of Ogre. Binaries are included at DependentExtensions\Ogre3DInterpDemo\DemoBinaries.zip

Related projects: ReplicaManager2 is used for object creation, destruction, and automatic serialization
