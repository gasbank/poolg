Project: PHPDirectoryServer

Description: A web-based directory server with corresponding C++ code to interface. The PHP page will show uploaded servers. Servers will stay online for a specified duration, and will be dropped if not refreshed.

Dependencies: Requires a PHP capable web-server, which includes most shared web servers

Related projects: None

Installation: Upload DirectoryServer.php to your web server, go to the page, press Reveal, and set the initial settings.
